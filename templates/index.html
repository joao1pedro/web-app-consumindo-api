Sobre

Esta API REST foi desenvolvida com o objetivo de armazenar modelos de Inteligência Artificial junto de suas descrições à uma base de dados para consultas futuras.

Foi utilizada a linguagem Python junto do framework Flask para construir a API e uma aplicação web para apresentar a API.

O quê é um modelo de machine learning?

Um modelo de machine learning é a expressão de um algoritmo que examina grandes volumes de dados para identificar padrões ou fazer previsões. Alimentados por dados, os modelos de ML são os mecanismos matemáticos da AI.
